12.1 Linux常用命令

1、发布dotnet程序#

dotnet publish "E:\xxx\ViperCenter.csproj" -c Release -r linux-x64 -o "E:\xxx\ViperCenter\bin"

2、jexus服务器#

重启服务器：

/usr/jexus/jws restart

3、连接mysql#

mysql -h 192.168.0.201 -P 3306 -u root -p123
### 4、查看端口占用
``` shell
netstat -ntlp
netstat -tunlp|grep 6660
yum install lsof
lsof -i:6660

5、查看后台进程#

ps -aux|grep ViperCenter

6、将目录dir1复制成目录dir2#

cp -R file1 file2

7、开放端口#

配置规则：

iptables -I INPUT -p tcp --dport 8080 -j ACCEPT

保存规则：

/usr/sbin/iptables-save

8、关闭端口#

/sbin/iptables -I INPUT -p tcp --dport 80 -j DROP 写入修改
/etc/init.d/iptables save 保存修改
service iptables restart 重启防火墙，修改生效

9、打包#

tar zcvf pmspace20170413.tar.gz pmspace

10、解压#

tar -zxvf pmspace20170413.tar.gz

11、查看进程的运行信息例如进程【3723】#

cd /proc/3723

Viper

Viper 是基于.NET平台下的Anno微服务框架的一个示例项目

快速入门

快速入门

Viper 简单易用，拉取项目代码、执行工程database目录下的数据库脚本SQL文件创建数据库、 修改 ViperService 数据库连接字符串，便可依次启动注册中心 ViperCenter、服务Provider ViperService, 、网关已经监控 Viper。

安全、稳定、高可用

Viper 是微服务快速开发框架Anno后期整理后开源核心代码下的一个模板工程， 底层通讯可以随意切换thrift grpc。 自带服务发现、调用链追踪、Cron 调度、限流、事件总线、CQRS 、DDD、类似MVC的开发体验，插件化开发。

全平台可监控

一个不可监控的微服务平台是可怕的，出了问题 难以准确定位问题的根源， Anno则提供了一套完整的监控体系，包括链路追踪、服务占用的系统资源、系统自身 CPU、内存、硬盘使用率实时可监控等等。

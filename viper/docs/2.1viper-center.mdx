### 2.1.1 一行代码的注册中心

 注册中心概述

<img src={useBaseUrl("img/viper01.jpg")} />

```java
using Anno.Rpc.Center;
using System;

namespace ViperCenter
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Title = "ViperCenter";
            Bootstrap.StartUp(args);
        }
    }
}

```
### 2.1.2 配置文件
配置文件 <b>Anno.config</b>
<b><code>Port</code></b>:为注册中心的监听端口
<br/>
<b><code>TimeOut</code></b>:为注册中心默认连接超时时间单位毫秒

```xml
<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <!--#lbs 配置-->
  <Port>7010</Port>
  <TimeOut>120000</TimeOut>
  <Servers>
 
  </Servers>
</configuration>
```
### 2.1.3 启动注册中心

配置完成后便可以启动注册中心

Windows
```shell
dotnet ViperCenter.dll
```

Linux
```shell
#!/bin/bash
if [ $1 == "start" ];then
nohup dotnet  ViperCenter.dll 2>&1 &
echo "$!" > pid
echo "start ok!"
elif [ $1 == "stop" ];then
kill `cat pid`
echo "stop ok!"
else
echo "Please make sure the position variable is start or stop."
fi

```
